Financial Pricing Error Prediction Using Signature Methods and Volatility Modeling

This project investigates whether modern machine learning techniques can improve the prediction of pricing errors in financial markets by combining traditional asset pricing models with advanced time-series representations.

View Notebooks GitHub Repository

Abstract

Traditional factor models such as the Fama-French Five-Factor Model explain much of the variation in expected stock returns, yet pricing errors (alphas) remain that may contain predictive information. This project investigates whether machine learning models can better identify these pricing errors by incorporating both sequential market dynamics and volatility information.

Pricing errors are estimated using the Fama-French Five-Factor model. Financial time series are then transformed into log-signature features, which capture higher-order sequential relationships, and combined with GARCH-based volatility statistics that model changing market risk. These feature sets are evaluated individually and jointly using Convolutional Neural Networks (CNNs), XGBoost, and ensemble learning methods on both daily and monthly datasets.

Results indicate that combining signature representations with volatility features provides richer information than traditional factor models alone. The hybrid feature representation consistently improves predictive performance and demonstrates the potential of rough path theory and modern machine learning techniques in quantitative asset pricing.

Research Motivation

Classical asset pricing models explain systematic risk but leave residual pricing errors that may reflect market inefficiencies or omitted information. The primary objective of this research is to determine whether these pricing errors can be predicted using advanced representations of financial time series.

Rather than relying solely on factor returns, this project incorporates both temporal market structure through log-signature transforms and dynamic volatility estimates through GARCH models.

Methodology

  • Estimate pricing errors using the Fama-French Five-Factor Model.
  • Generate log-signature features from rolling return sequences.
  • Estimate conditional volatility using GARCH models.
  • Create three feature sets:
    • Signature Features
    • GARCH Features
    • Combined Signature + GARCH Features
  • Train CNN, XGBoost, and ensemble learning models.
  • Evaluate models using Accuracy, F1 Score, ROC-AUC, and portfolio ranking performance.

Key Findings

  • Log-signature features effectively capture nonlinear temporal structure in financial time series.
  • GARCH volatility statistics contribute meaningful information regarding changing market conditions.
  • The combined feature representation generally outperforms either feature type alone.
  • Machine learning models successfully identify pricing error patterns that are difficult to capture using traditional econometric approaches.
  • The resulting framework shows promise for identifying assets with relatively high expected future returns.
